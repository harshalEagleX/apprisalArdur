## General Requirements

| Item | Rule |
|------|------|
| **Invoice** | No invoice should be included in the report |
| **Form Type** | Report form type must match what was ordered |
| **Transaction Type** | Must match the loan type on the order form |
| **Trainees** | Trainees are NOT allowed to sign any reports; if they are not licensed/certified, the report must be sent back for correction |

---

## Subject Section

| Item | Requirement |
|------|-------------|
| **Property Address** | Must match the Client Engagement Letter exactly. Verify with USPS. If USPS verification differs, a comment must be provided |
| **City, State, Zip Code, County** | Must match the Client Engagement Letter exactly |
| **Borrower** | Must match Client Engagement Letter exactly, watch for commonly missed co-borrower names and spelling errors. Does not need to match Owner of Public Record. If refinance and owner differs from borrower, a comment is required |
| **Owner of Public Record** | Provided by the appraiser and be current. If refinance and owner differs from borrower, a comment must be provided |
| **Legal Description** | Cannot be blank |
| **Assessor's Parcel # (APN)** | Cannot be blank |
| **Tax Year** | Must be current or within the last 2 years; cannot be blank |
| **R.E. Taxes** | Cannot contain decimals; cannot be blank |
| **Neighborhood Name** | Must provide the subdivision name or most common area name. Cannot be blank, None, N/A, or Unknown |
| **Map Reference** | Must be provided and current; must be numeric |
| **Census Tract** | Must be provided in format: four digits, then two digits after decimal |
| **Occupant** | If tenant: verify lease dates and amount. If vacant: must state if utilities are on. Photos should match occupancy status |
| **Special Assessments** | Must comment if any exist (amount and purpose). If none, enter "0" |
| **PUD and HOA** | If HOA dues are mandatory, it's a PUD. Proper information should be completed in the PUD section. Mark "per year" or "per month" |

### Subject History (Prior Sales/Listings)

| Scenario | Action Required |
|----------|-----------------|
| **No prior listings or current sale** | Mark "No" and include data source (abbreviated MLS name, e.g., MXLMLS) |
| **Currently listed or has a sale** | Provide on Page 1 in format: `DOM #; Abbreviated MLS name and MLS #, Current list/sale price, Current list/sale date` |
| **Listed but NOT a purchase** | Comment required if appraised value varies from most recent listing price by >3% |

## Contract Section

> **For Purchase Transactions Only** — If not a purchase, this entire section must be blank.

| Item | Requirement |
|------|-------------|
| **Did/Did not analyze contract** | Contract must be analyzed |
| **Commentary** | Must show analysis of contract, sale type (Arms-Length, non Arms-Length, REO sale, etc.) and reconcile contract price with market value opinion (higher or lower) |
| **Contract Price** | Must be provided from the Contract |
| **Date of Contract** | Must be the date of last signature (fully executed date). Example: Seller signs 3/1/2019, Buyer signs 4/2/2019 → Contract date is 4/2/2019 |
| **Owner of Record and Data Source** | Must check yes or no. If no, commentary required |
| **Financial Assistance** | Comment if applicable |
| **Personal Property** | Identify all items of personal property in the contract and state whether they contribute to value |

## Neighborhood Section

| Item | Requirement |
|------|-------------|
| **Characteristics** | Provide trends and land use. Built-up percentage must match Present Land Use |
| **Housing Trends** | Be aware of characteristics and land use. If DECLINING or INCREASING, verify proper directional adjustments in sales grid. Trend should match MC form or specific commentary is required |
| **Unit Housing Price and Age** | Unadjusted comparable sales prices must fall within the ranges (comment required if not in neighborhood). Comment required if market value differs from predominant by >10% |
| **Present Land Use** | Must complete. Identify "other" land uses if any percentage given. "Vacant" is not acceptable — must be specific |
| **Boundaries** | Must list North, South, East, West only (no abbreviations). Boundaries must be visible on location map |
| **Description** | Must be specific to the area (no canned commentary) |
| **Market Conditions** | Must be provided. Stating "See 1004MC" is NOT acceptable |

---

## Site Section

| Item | Requirement |
|------|-------------|
| **Dimensions** | Must list site dimensions (e.g., 50 X 100). If irregular and dimensions cannot be provided, plat map with subject clearly marked is required |
| **Area** | Must be in square feet (e.g., 9546 sf). If over 1 acre (43,560 sf), must be in acreage. If on acreage with barn/large outbuildings and zoned Agricultural, comment if subject is a working farm |
| **Shape** | Must be provided. If irregular and dimensions not included, plat map required |
| **View** | Must be UAD Compliant (Rating; factor; other as needed). Must match grid section |
| **Zoning Classification, Description, Compliance** | If no zoning or legal non-conforming, state whether subject can be rebuilt to current use if destroyed >50%. If non-conforming, state why (site size/width, improvement type, illegal use) |
| **Highest and Best Use** | If existing use is not highest and best use, notify. **Highest and best use analysis must be provided** |
| **Utilities/Off-Site Improvements** | If private well/septic: state if typical, if public access possible, and effect on marketability. If private street: state if road maintenance agreement exists, who pays, and effect on marketability. **If a box is checked, description cannot state "None" or "N/A"** |
| **Flood Hazard Area** | If in FEMA special flood hazard area, state if it affects marketability. Comment on which comparables, if any, are in a flood zone |
| **Utilities and Off-site Typical** | Check yes or no. If no, commentary required |
| **Adverse Site Conditions** | Check yes or no. If yes, commentary required. If external obsolescence (commercial influence, traffic street, airport, high-tension wires, etc.), explain impact on marketability and provide comparables with similar obsolescence |

## Improvement Section

| Item | Requirement |
|------|-------------|
| **Foundation** | All fields must be completed as applicable. Compare to grid section |
| **Exterior Description** | All fields must be completed as applicable. Compare to grid section |
| **Interior** | Complete as applicable. If car storage is "none," all # of car fields must be 0 and driveway surface blank. Do not mark "none" if subject has a driveway |
| **Utilities** | Must state if utilities were on |
| **Appliances** | Note only built-in items |
| **Above Grade Contains** | Exclude partially or fully below-grade areas. Below-grade beds/baths must not be included. Must match grid section |
| **Additional Features** | List energy efficient items. If none, state "NONE" |
| **Condition of Property** | Must be UAD Compliant (e.g., C3, C4). Must match photos, commentary, and be supported by effective age |
| **Adverse Conditions** | If "Yes" checked, confirm items noted truly affect livability. If not, may be a typo needing correction |
| **Does Property Conform to Neighborhood** | Yes or No. If No, provide extensive commentary: Why? Can it be rebuilt if destroyed >50%? Is it overbuilt? |

## Sales Comparison Approach

### Comparable Selection Rules

| Value Range | Minimum Required |
|-------------|------------------|
| **Less than $1 million** | 3 sales + 2 listings |
| **$1 million or more** | 4 sales + 2 listings |
| **All reports** | At least 1 active/pending sale (unless 0 listings stated at top of page 2) |

### Grid Requirements

| Field | Requirement |
|-------|-------------|
| **# of Comparable Properties Currently Offered** | Include only competing listings. Range must match 1004MC form for current to 3 months |
| **# of Comparable Sales within 12 Months** | Include only competing sales. Range must match 1004MC form and predominant price ranges from page 1 |
| **Address** | Must be UAD compliant and USPS verified. Subject address must match page 1 exactly |
| **Proximity to Subject** | Must be ≥0.01 miles with proper direction (N, S, NW, NE, etc.). If blank, verify location map is provided |
| **Data Sources** | UAD Compliant: Specific MLS name + MLS number (e.g., MISMLS#3546935;DOM12). DOM must be provided or "Unk" (with commentary if unknown). DOM should reflect marketing time on page 1 |
| **Verification Sources** | At least 1 must be provided with specific source in full (e.g., Orange County Assessor Tax Card) |
| **Sale or Financing Concessions** | UAD Compliant: Sale type, financing type, concession. Verify adjustments are in appropriate direction. If no adjustment, field should contain "0" |
| **Date of Sale/Time** | UAD Compliant: Status type, sale date, contract date. Contract date is ALWAYS before sale date. Be aware of housing trends. Commentary needed for 90 days, 6 months, or >12 months |
| **Location** | UAD Compliant (Rating;type;other). Max 2 descriptors, semi-colon separated. If no adjustment, enter "0" |
| **Leasehold/Fee Simple** | Must be provided. If leasehold, provide similar comps. Fannie and Freddie allow fee simple sales |
| **Site** | <1 acre = SF; ≥1 acre = acreage. If no adjustment, enter "0" |
| **View** | UAD Compliant (Rating;factor;factor2). Max 2 descriptors. If waterfront, report frontage in linear feet on an extra line item |
| **Design (Style)** | UAD Compliant: Attachment type, # of stories, Design style. If no adjustment, enter "0". No adjustment needed for style difference |
| **Quality of Construction** | UAD Compliant (Q3, Q4, etc.). If no adjustment, enter "0" with commentary |
| **Actual Age** | UAD Compliant. Verify actual age matches year built. If no adjustment, enter "0" |
| **Condition** | UAD Compliant. Verify matches photos. If no adjustment, enter "0" |
| **Above Grade Room Count/GLA** | Must be provided. Verify rooms and GLA match sketch |
| **Basement & Finish/Rooms Below Grade** | UAD Compliant. Must match page 1 and complete exit type. If no adjustment, enter "0" |
| **Functional Utility** | If no adjustment, enter "0" |
| **Heating/Cooling** | Must match page 1 information |
| **Garage/Carport** | UAD Compliant: # of cars and type. If no adjustment, enter "0" |
| **Porch/Patio/Deck** | Must match page 1 or noted as "none." If no adjustment, enter "0" |
| **Net Adjustment (total)** | Form calculates. Verify adjustments are in correct direction |
| **Adjusted Sale Price** | Form calculates. Verify correct direction. Appraisers should round adjustments |

### Bracketing Rule

All fields must be bracketed. If unable to bracket, extensive commentary must be provided explaining why.

### Sales History

| Item | Requirement |
|------|-------------|
| **Research sale/transfer history** | Must be marked "Yes" |
| **Subject 3-year history** | Required with data source. MLS name must be specified (e.g., MFRMLS, not just "MLS") |
| **Comparable 12-month history** | Required with data source. MLS name must be specified |
| **Sales History Table** | Date in mm/dd/yyyy format. If none, leave blank. Only most recent should be reported; if multiple, provide commentary |
| **Price of Prior Sale/Transfer** | Only most recent figure. If multiple, provide commentary |
| **Data Source(s)** | Required even if no sale. Public record requires Doc #. MLS requires MLS name + # (e.g., MFRMLS #T2321256) |
| **Effective Date of Data Source(s)** | Required even if no sale or transfer reported |

### Listings

| Requirement |
|-------------|
| Give listings a list-to-sales price ratio adjustment to reflect potential sales price, or explain why no adjustment is warranted. Listings must support the opinion of value |

### Summary of Sales Comparison Approach

- Discuss comparables specifically
- Address distant comparables and adjustments given (or lack thereof)
- Opinion of value should be bracketed by unadjusted AND adjusted sales prices
- If unable to bracket, provide sufficient commentary

## Reconciliation

| Item | Requirement |
|------|-------------|
| **Indicated Value by Sales, Cost, Income** | Sales must match market value (exception: income approach). Cost Approach must be provided if completed. Income approach if applicable |
| **Commentary** | Discuss how sales were weighted. Explain why Cost/Income approaches were not developed if applicable |
| **Terms to AVOID** | "Equal weight/consideration given to all comparables" and "Weighted average" |
| **Appraisal: As Is or Subject To** | Verify appropriate box is checked and commentary supports. If subject to repairs, provide itemized list with cost to cure (e.g., "New Carpet, $3,000; Paint throughout, $1,000; Closet door, $100") |

## Cost Approach

| Item | Requirement |
|------|-------------|
| **Site Value** | Required in every report |
| **Support for Site Value Opinion** | State method(s) used (land sales comparison, market extraction, or allocation). If site value >30% of market value, state and explain. Use current date for cost approach data |
| **Variance from Appraised Value** | If Cost Approach varies >20% from appraised value, provide detailed explanation |
| **If Not Developed** | Include statement: *"The Cost Approach was considered but not developed in this appraisal as it was determined to not be necessary to develop a credible opinion of value."* |

> **Note:** Cost approach is required on all USDA loans.

## Income Approach

| Item | Requirement |

## Signature Page

| Item | Requirement |
|------|-------------|
| **Appraiser Signature** | Report must be signed |
| **Name** | Must read as it is on ASC.gov |
| **Company Name/Address/Phone/Email** | Should match engagement letter |
| **Date of Signature** | Original sign date; update each time a revision is made |
| **Effective Date of Appraisal** | Must be same as inspection date. This date should NEVER change |
| **State Certification/License #** | Must read as on ASC.gov. Verify product type can be completed by State Licensed appraiser if applicable |
| **State** | Must match subject property state AND appraiser's license state attached |
| **Expiration Date of License** | Must be valid and match attached license |

| **Email Address** | Not required |

## Addendum Page

| Item | Requirement |
|------|-------------|
| **Commentary** | Must be specific to the report. **No canned commentary.** Headers should ALWAYS be used (Beginning addendum, Comments on sales comparison, etc.) |
| **Comparable Selection** | Provide detailed reasoning behind selection. Commentary should state WHY, not just what was done |
| **Dated Sales** | Address dated sales, distant comparables, comparables crossing major boundaries, and need for comparables of various designs/age/GLA |

## Market Conditions Addendum (1004MC Form)

> **Note:** For Conventional loans, 1004MC is not required. FHA/USDA are the only ones requiring this form.

| Item | Requirement |
|------|-------------|
| **Inventory Analysis** | All lightly shaded areas required or comment why not. Blank spaces = "0" or N/A |
| **Total # of Comparable Sales (Settled)** | Must total to data at top of sales grid for subject neighborhood |
| **Total # of Comparable Active Listings** | Should match top of sales grid for currently offered properties. Current-3 months must match |
| **Median Sale & List Price, DOM, Sale/List %** | All lightly shaded areas required. Blank = "0" or N/A. Dark shaded = N/A if cannot be provided |
| **Overall Trend** | Mark trends where at least 2 data points exist. If declining/increasing, apply appropriate market conditions adjustment or explain why not applicable |
| **Commentary** | Must be completed for each section. Must be specific, no canned commentary |
| **Condo/Co-Op Projects** | All lightly shaded areas must be completed for Condo/Co-Op, specific to subject's project |

## Special Situations

### Additions to the Subject

If an addition is present, describe and comment on:
1. Is the addition permitted?
2. Does it conform to original structure in quality and functional utility?
3. Impact on marketability and compliance with local zoning (including consequences of non-compliance)

> **If addition creates health/safety issue:** Appraisal should be subject to remediation or removal. Cost to cure must be provided.

### Security Bars

- Potential safety issue
- Comment whether they have safety release latches
- Comment if they meet building codes
- If meeting local codes, acceptable regardless of release latches

### Square Footage

- Cannot use below-grade SF in gross living area unless necessary
- If necessary (MLS/public records lump above and below grade together, or subject market views below grade as GLA), explain why and ensure comparables reflect the same

### Design

- If unique (green home, log home, 1 bedroom, accessory unit, etc.), include similar comparables
- If not available, explain in detail why

## FHA Assignments

> **Important:** If property does NOT meet minimum HUD guidelines, report MUST be made subject to.
>
> Per HUD's directive, all Site Condos must be on the 1073 form.

### FHA-Specific Requirements

| Item | Requirement |
|------|-------------|
| **FHA Case #** | Must be in upper right corner on ALL pages. Format: `XXX-XXXXXXX` (10 numbers total with dash) |
| **Intended User** | Must include: *"The INTENDED USER of this appraisal report is the Lender/Client and HUD/FHA. The Intended Use is to evaluate the property that is the subject of this appraisal for a mortgage finance transaction, subject to the stated Scope of Work, purpose of the appraisal, reporting requirements of this appraisal report form, and Definition of Market Value. No additional Intended Users are identified by the appraiser."* |
| **Intended Use** | Must include: *"The INTENDED USE of this appraisal is to support FHA's decision to provide mortgage insurance on the real property that is the subject of the appraisal."* |
| **FHA/HUD Minimum Property Requirements** | Include statement if subject meets FHA MPR per HUD Handbook 4000.1, or if "subject to" repairs, state property will meet upon completion |
| **Comparable Sales** | Comps 1, 2, and 3 MUST be within 12 months of effective date |
| **FHA Repairs** | If repairs surpass cosmetic level, complete "subject to" with itemized list and cost to cure. Includes making water, furnace, electrical, and mechanics operational if not at inspection |
| **Primary Heat Source = Space Heater** | Verify: (1) Permanently affixed, (2) Can heat/maintain ≥50°F in all living spaces and spaces with plumbing, and/or acceptable to local authority |
| **Security Bars** | Clearly state at least one window in every bedroom has quick-release latches for emergency exit |
| **Photo Requirements** | ALL sides: Front, Rear, Left side, Right side |
| **Estimated Remaining Economic Life** | Required for all single-family, multi-family, and condos. If <30 years, clearly explain why subject doesn't meet FHA minimum |
| **Attic/Crawl Requirement** | Inspection/viewing required WITH PHOTOS. Include comment stating "Head and Shoulders inspection" was made |
| **Well and Septic** | Comment if public water/sewer available. If available, provide cost to hookup |
| **Sketch** | All outbuildings, garages, and structures (deck, porch, patio, balcony) must be labeled with dimensions showing covered or uncovered |
| **Appliances** | MUST operate all built-in appliances (stove, dishwasher, etc.). Statement required that appliances were operational or not |

## Quick Reference: Common Rejection Reasons

| Issue | Standard Rejection Language |
|-------|---------------------------|
| **Address mismatch** | "Property address does not match with order form." |
| **City mismatch** | "Property City name does not match with order form." |
| **Zip code mismatch** | "Property Zip code does not match with order form. (First 5 digits must match)" |
| **Missing co-borrower** | "Please include Co-borrower name _______ as per order form." |
| **Owner/Borrower mismatch (Refinance)** | "Assignment type is noted as 'Refinance'; however, the owner name and borrower name are different, please revise or comment." |
| **N/A neighborhood name** | "The neighborhood name in subject section is mentioned as N/A. Per UAD requirements, the appraiser should enter a recognized neighborhood name or common location name." |
| **Occupancy/photos mismatch (Owner marked, looks vacant)** | "The subject section indicates owner occupied; however, photos appear to show vacant. Please revise or comment." |
| **Occupancy/photos mismatch (Vacant marked, looks occupied)** | "Subject section occupancy marked as vacant; however, photos appear occupied. Please revise or comment if it is a staged home." |
